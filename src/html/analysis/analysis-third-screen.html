Анализ требований с построением Use Case модели:

Выявляют функции программной системы, которые отвечают за бизнес-логику и обеспечивают трассируемость функциональных требований.
Детально описывают потоки событий в каждом из сценариев бизнес-логики.
Выстраивают Use Case-модель программной системы.
Используя инструменты прототипирования Axsure и Balsaminq, создают эффективные прототипы экранных форм.
Проверяют сценарии и прототипы экранных форм на соответствие требованиям заказчика.
